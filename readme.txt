== Description ==

This plugin which is simple in its funcationality and in its name(haha). This plugin provide a simple and aesthetic list of plugins and themes that are installed in your site. This plugin shows there current version and if there is any update present for given wp-contents (represent both themes and plugins ). This is first iteration of this plugin so, fasten your seat belt 
because with every new update you would feel the thrust.
